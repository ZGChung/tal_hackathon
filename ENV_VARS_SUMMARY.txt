========================================
ENVIRONMENT VARIABLES FOR RENDER
========================================

After deploying on Render, copy these into your dashboard:

BACKEND SERVICE:
----------------
DEEPSEEK_API_KEY=YOUR-DEEPSEEK-API-KEY-HERE
ALLOWED_ORIGINS=https://YOUR-FRONTEND-URL.onrender.com
  ↑ Replace with your actual frontend URL from Render

FRONTEND SERVICE:
-----------------
REACT_APP_API_URL=https://YOUR-BACKEND-URL.onrender.com
  ↑ Replace with your actual backend URL from Render

========================================
STEPS:
1. Deploy services on Render
2. Get your backend URL (e.g., tal-hackathon-backend-xyz.onrender.com)
3. Get your frontend URL (e.g., tal-hackathon-frontend-abc.onrender.com)
4. Set ALLOWED_ORIGINS = your frontend URL
5. Set REACT_APP_API_URL = your backend URL
6. Set DEEPSEEK_API_KEY = YOUR-DEEPSEEK-API-KEY-HERE
7. Redeploy both services
========================================
