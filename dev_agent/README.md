# Development Agent Tasks

This folder contains task summaries and prompts for initializing different agents in a multi-agent development workflow.

## Structure

Each agent task should have:
- A task summary file (e.g., `agent_frontend_task.md`)
- An initialization prompt file (e.g., `agent_frontend_prompt.md`)

## Agent Roles

- **Frontend Agent**: Handles React frontend development
- **Backend Agent**: Handles FastAPI backend development
- **Testing Agent**: Handles test implementation and TDD
- **Integration Agent**: Handles integration between frontend and backend
